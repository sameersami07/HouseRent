# HouseHunt Client

This is the frontend for HouseHunt, a rental marketplace built with React, Vite, Material UI, Bootstrap, and Axios.

The app includes:

- Responsive dashboards for Renters, Owners, and Admin.
- Advanced property search filters for location, price, property type, furnishing, and amenities.
- Booking request form with renter details and owner contact preview.
- Owner property management with add/edit/delete and status controls.
- Admin panel for approving owner accounts and overseeing listings.
- Embedded Google Maps preview in property detail views.

## Run the frontend

```bash
cd Client
npm install
npm run dev
```

## Notes

- The frontend communicates with the backend at `http://localhost:8000`.
- Start the server before using the app.
- Demo accounts and sample property listings are available for testing.
